###和前端相关：

1、先定义组件 然后安装到项目使用

2、组件 props 类型要声明

  • array React.PropTypes.array //数组
  • bool React.PropTypes.bool.isRequired //布尔值 而且是必须的
  • func React.PropTypes.func //函数
  • number //数字
  • object //对象
  • string //字符串
  • any //任何类型 number string elements array
  • arrayOf //
  • element //React 元素
  • instanceOf
  • node
  • objectOf
  • oneOf
  • oneOfType
  • shape

3、package.json name 定义 必须是以下三个组下

  • @ht,
  • @httest,
  • @person

举例：定义userInfo的组件

{
   'name' : '@ht/userInfo'
    ...
 }
